Como elaborar un código prolijo!

Requerimientos

  • Saber el lenguaje en el que se programa
  • Entender la sintaxis
  • Dejar identado el código
    • Se puede configurar el editor de código
  • Realizar comentarios en el código
  • Ordenar los archivos en las carpetas correspondientes

Pasos para guardarlo

  1. Abrimos el editor de código
  2. Pulsamos sobre nuevo
  3. Pulsamos Guardar
  4. Localizamos la carpeta donde corresponde guardar el archivo
